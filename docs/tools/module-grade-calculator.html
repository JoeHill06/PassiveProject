Understanding Module Grade Calculations

Common Assessment Weightings

Different modules use different assessment structures. Common weightings include:

  • 60% Exam / 40% Coursework: Most common in UK universities
  • 70% Exam / 30% Coursework: Common for science and maths modules
  • 50% Exam / 50% Coursework: Equal weighting
  • 100% Coursework: No exam (dissertations, projects)
  • 100% Exam: No coursework

How Module Grades Are Calculated

Your final module grade is calculated using the formula:

Final Grade = (Coursework Mark × Coursework Weight) + (Exam Mark × Exam Weight)

For example, if you got 75% in coursework (40% weight) and 65% in the exam (60% weight):

Final Grade = (75 × 0.4) + (65 × 0.6) = 30 + 39 = 69%

Frequently Asked Questions

What exam mark do I need to get a First?

This calculator will tell you exactly what you need! Enter your coursework marks and set your target grade to 70%, and it will calculate the required exam mark.

What if I have multiple coursework components?

Add each coursework component separately with its individual mark and weighting. For example, if Essay 1 is worth 20% and you got 75%, add it as: Mark=75%, Weight=20%.

What if coursework weightings don't add up to the total coursework weight?

Make sure your individual coursework component weightings add up to the total coursework weighting. For example, if coursework is 40% total, your individual components should add up to 40%.

Is it possible to still get a First if I did poorly on coursework?

It depends on how the module is weighted. If the exam is worth 60% or more and you get 100% on it, you might still be able to achieve a First even with moderate coursework marks. Use this calculator to find out!

What if the required exam mark is over 100%?

If the calculator shows you need more than 100% on the exam to reach your target, unfortunately that target grade is no longer achievable. You may need to adjust your target to something more realistic.
